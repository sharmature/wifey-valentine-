## Quick Start

1. **Add your photo** (IMPORTANT!):
   - Place a photo of you two in `public/our-photo.jpg`
   - Make it square (800x800px or larger works best)
   - The puzzle will use this photo automatically

2. **Run the dev server**:
```bash
npm run dev
```

3. **Access the site**:
   - **Demo Mode (for testing)**: Add `?demo=true` to the URL
     - Example: `http://localhost:5173/?demo=true`
     - This lets you test ALL days with a day selector
   - **Real Mode (for her)**: Just the normal URL
     - Example: `http://localhost:5173/`
     - Shows only today's day, locks future days with countdown
